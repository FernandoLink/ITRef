
##### Estruturas de Repetição
***

**while**
```
while (contador<4) {
	System.out.println(contador);
	contador++;
}
```

***

**do while**
```
do {
	System.out.println(contador);
	contador++;
} while (contador<4);
```

***

**for**
```
for(int i=0;i<4;i++) {
	System.out.println(i);
}
```

***
**Obs.:**
* Comando *continue* interrompe o fluxo e retorna para o próxima iteração da repetição.
* Comando *break* sai da repetição.
