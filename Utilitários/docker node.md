
##### docker node
***


|Comando|Descrição|
|:--|:--|
|docker node --help| Mostrar ajuda dos comandos do node|
|docker node ls| Listar nodes no swarm|
|docker node promote \<hostname\>| Promove o node para manager|
| docker node demote \<hostname\>|Despromove o node de managar|
|docker node rm \<hostname\>|Remover o node|
|docker node update | Atualizar metadados sobre um node, tais como sua disponibilidade, rótulos ou papéis|
|docker node inspect| Retorna informações sobre o node|
